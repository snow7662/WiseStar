# WiseStar-MathAgent Frontend - 路由文档

## 路由映射表

本项目使用 React Router v6 的 HashRouter 模式，所有路由均以 `#` 开头。

### 完整路由列表

| 路由路径 | 页面名称 | 组件文件 | 说明 |
|---------|---------|---------|------|
| `/` | 根路径 | - | 自动重定向到 `/dashboard` |
| `/dashboard` | 仪表盘 | `src/pages/Dashboard.jsx` | 系统首页，展示功能概览 |
| `/solve` | 智能解题 | `src/pages/SolveProblem.jsx` | RePI解题功能页面 |
| `/generate` | 题目生成 | `src/pages/GenerateQuestion.jsx` | QuestionGeneration出题页面 |
| `/memory` | 学习记忆 | `src/pages/LearningMemory.jsx` | Memory学习历史查询页面 |
| `/statistics` | 统计分析 | `src/pages/Statistics.jsx` | 数据可视化统计页面 |
| `/daily` | 每日一题 | `src/pages/DailyQuestion.jsx` | 每日推荐题目页面 |
| `/plot` | 几何画图 | `src/pages/GeometryPlot.jsx` | 解析几何图形绘制页面 |

### URL访问示例

假设应用运行在 `http://localhost:3048`，则完整URL格式为：

- 仪表盘: `http://localhost:3048/#/dashboard`
- 智能解题: `http://localhost:3048/#/solve`
- 题目生成: `http://localhost:3048/#/generate`
- 学习记忆: `http://localhost:3048/#/memory`
- 统计分析: `http://localhost:3048/#/statistics`
- 每日一题: `http://localhost:3048/#/daily`

## 路由配置说明

### 路由定义位置

路由配置在 `src/App.jsx` 文件中：

```jsx
<Routes>
  <Route path="/" element={<Navigate to="/dashboard" replace />} />
  <Route path="/dashboard" element={<Dashboard />} />
  <Route path="/solve" element={<SolveProblem />} />
  <Route path="/generate" element={<GenerateQuestion />} />
  <Route path="/memory" element={<LearningMemory />} />
  <Route path="/statistics" element={<Statistics />} />
  <Route path="/daily" element={<DailyQuestion />} />
</Routes>
```

### 导航菜单配置

导航菜单配置在 `src/components/Layout.jsx` 文件中：

```jsx
const navigation = [
```

## 开发说明

### 启动开发服务器

```bash
cd frontend
anpm install
anpm run dev
```

服务器将在 `http://localhost:3048` 启动。

### 构建生产版本

```bash
anpm run build
```

构建产物将输出到 `dist/` 目录。

## 注意事项

1. **必须使用HashRouter**：项目使用HashRouter而非BrowserRouter，确保在WebC环境中正常工作
2. **端口固定为3048**：与WebC环境配置保持一致
3. **所有路由必须可直接访问**：每个路由都应该支持直接通过URL访问，不依赖前置导航
4. **响应式设计**：所有页面都支持移动端和桌面端访问
